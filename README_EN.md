# The Construction Lead Machine - Premium Landing Page

## 🔥 Product Description
This is a **premium landing page** for construction and renovation companies that want to **capture clients while they sleep**.  
Designed to turn visitors into qualified leads with **high-impact UX**, frictionless and fully mobile-friendly.

## 🎯 Problem It Solves
Many renovation businesses lose clients because:
- They lack a professional website
- Forms are complicated
- No follow-up when a visitor shows interest

This template converts any visitor into a potential client **with a single click**.

## 💎 Features
- Hero section with direct WhatsApp + Email CTA
- Services section with “Get a Quote” buttons
- Global contact modal (omnichannel) triggering WhatsApp + Email
- Projects section (6 completed works gallery)
- Professional footer with contact and branding
- **Dark + High-End design**, responsive and fast

## 🛠️ Stack
- React.js + Vite
- Tailwind CSS
- No backend – all functions via WhatsApp + Email

## 🧩 Setup
1. Clone the project.
2. Install dependencies: `npm install`
3. Update `CONFIG.js` with your data:
   - Client name
   - Phone
   - Services
4. Deploy to any free hosting (Vercel recommended)
5. Done! Your website is live and capturing leads automatically.
