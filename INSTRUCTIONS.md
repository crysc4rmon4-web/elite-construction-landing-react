# 🚀 Quick Start Guide - Construction Lead Machine

Thanks for your purchase! Follow these steps to launch your site:

### 1. Installation
Extract the ZIP and run:
`npm install`

### 2. Personalization (The "Magic" Step)
Open `src/config/data.js`. This is the only file you need to edit. 
Change the `brand`, `contact`, `services`, and `projects` objects with your own info.

### 3. Images
Use links from **Unsplash.com** or **Pexels.com**. The system will automatically resize and compress them for you to ensure maximum speed.

### 4. Deployment
We recommend **Vercel** or **Netlify**:
1. Connect your GitHub repo or drag and drop the folder.
2. Build command: `npm run build`
3. Output directory: `dist`
